# AI Hand Control & Sign Language Writer

A computer vision project using **OpenCV** and **MediaPipe** to control system windows and translate sign language into text.

## Current Features
* Real-time hand landmark detection (21 points).
* Optimized FPS tracking (Target: 30+ FPS).
* Mirrored camera feed for intuitive control.

## Roadmap
- [ ] Gesture-based window control (Close/Minimize).
- [ ] Virtual Mouse movement.
- [ ] Sign Language alphabet recognition (ML Classifier).
